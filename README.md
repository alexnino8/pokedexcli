Pokedex CLI
A command-line Pokedex application built in Go.

Current Progress
I am currently building out the core infrastructure of the Pokedex. As of now, the REPL (Read-Eval-Print Loop) is fully functional and supports basic command processing.

Supported Commands
exit: Gracefully shuts down the Pokedex.
help: Prints a description of how to use the Pokedex and lists available commands.
Installation
Ensure you have Go installed on your system.

Clone the repo: git clone https://github.com/yourusername/pokedexcli.git
Run the application: go run .
